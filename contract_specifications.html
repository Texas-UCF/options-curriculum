Each option has several specifications which describe the contract. Here are the common ones:
  • Type: call or put
  • Underlying: product that will trade if option is exercised
  • Strike: price at which underlying will trade if option is exercised
  • Expiration: final time at which the option can be exercised
  • Multiplier: amount of the underlying that will trade if option is exercised
Suppose I own an option that gives me the right to sell 100 boxes of pizza for $8 on June 27, 2020. Here are the specifications broken down
  • Type: put (the option gives the right to sell)
  • Underlying: box of pizza
  • Strike: $8
  • Expiration: June 27, 2020
  • Multiplier: 100
Another specification is the exercise condition which describes when the option can be exercised to trade the underlying. There are two common choices:
  • European: option can only be exercised on expiration day
  • American: option can be exercised anytime up to or on expiration day
The example put option above on the pizza boxes was European because we only had the right to sell on June 27, 2020. If we had the right to sell anytime from until June 27, 2020, it would be American. The final specification we will discuss is the settlement method which describes what is exchanged between the two parties if the option is exercised. There are two common choices:
  • Physical: the actual underlying is traded for money
  • Cash: ...it's complicated
Using the example put option above, if we actually sell 100 boxes of pizza and receive $800 when we exercise the option, then it uses physical settlement. If instead we decide to just exchange the cash difference between the strike and a cost of a box of pizza today, it would be cash settled. Cash settlement is useful when the underlying is difficult (or impossible) to hold. We'll revisit this settlement method and give a better explanation with examples when we discuss exercising options. 		We usually talk about the expiration date, but technically the expiration time should also be specified. Regardless of the multiplier, the strike is the price at which one of the underlying will trade. So if the option is exercised, the amount of money that is given is actually strike * multiplier. The exercise condition names don't actually suggest any geographical meaning. You can find both European and American options everywhere.
